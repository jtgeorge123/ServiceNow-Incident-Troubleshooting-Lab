ServiceNow Incident Management Lab

A hands‑on incident‑management training environment built using a free ServiceNow Personal Developer Instance. This project simulates real enterprise ITSM workflows with Incidents, Assignment Groups, SLAs, Work Notes, CI mapping, and triage procedures.

This lab demonstrates practical IT support and troubleshooting skills aligned with IT Support Associate II, Systems Administration, and SOC Analyst responsibilities.
